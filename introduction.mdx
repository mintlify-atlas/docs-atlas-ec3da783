## Welcome

This guide walks you through a complete installation of **Frappe** and **ERPNext** on Ubuntu (18.04, 20.04, 22.04) and ArchLinux. Whether you're setting up a local development environment or a production server, this guide has you covered.

**Frappe** is an open-source, full-stack web application framework built on Python and JavaScript. **ERPNext** is a world-class open-source ERP system built on top of the Frappe framework.

## Technology Stack

| Component | Version | Purpose |
|-----------|---------|---------|
| Python | 3.6+ | Server-side language |
| Node.js | 20.x | JavaScript runtime |
| MariaDB | 10.3.x | Primary database |
| Redis | 5+ | Caching and queues |
| Nginx | Latest | Reverse proxy |
| Supervisor | Latest | Process manager |
| wkhtmltopdf | 0.12.5 | PDF generation |
| Yarn | 1.12+ | JavaScript package manager |
